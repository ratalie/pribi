## Documentación de `@registros`

Este directorio centraliza la información del trabajo en curso para el nuevo dominio `@registros`, independiente de `@registro-societario`.

- `architecture/registro-sociedades.md`: Lineamientos de DDD + arquitectura hexagonal, capas y flujo de datos.
- `roadmaps/registros.md`: Roadmap vivo con estado actual, próximos pasos y registro de cambios.

> Mantener estos documentos actualizados cada vez que se avance o se definan decisiones clave. Esto permitirá iterar sin afectar módulos existentes y facilitará la transición cuando el nuevo dominio reemplace al legado.
