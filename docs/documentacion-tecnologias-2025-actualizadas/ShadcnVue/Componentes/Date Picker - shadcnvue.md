## Date Picker

A date picker component with range and presets.

[Component Source](https://github.com/unovue/shadcn-vue/tree/dev/apps/www/src/registry/default/examples/DatePickerDemo.vue) [API Reference](https://www.reka-ui.com/docs/components/calendar.html)

## Installation

The Date Picker is built using a composition of the `<Popover />` and either the `<Calendar />` or `<RangeCalendar />` components.

See installations instructions for the [Popover](https://www.shadcn-vue.com/docs/components/popover.html), [Calendar](https://www.shadcn-vue.com/docs/components/calendar.html), and [Range Calendar](https://www.shadcn-vue.com/docs/components/range-calendar.html) components.
