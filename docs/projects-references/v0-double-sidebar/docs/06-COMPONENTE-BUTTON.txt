CÓDIGO COMPLETO

```typescript
import * as React from 'react'
import { Slot } from '@radix-ui/react-slot'
import { cva, type VariantProps } from 'class-variance-authority'

import { cn } from '@/lib/utils'

const buttonVariants = cva(
  "inline-flex items-center justify-center gap-2 whitespace-nowrap rounded-md text-sm font-medium transition-all disabled:pointer-events-none disabled:opacity-50 [&_svg]:pointer-events-none [&_svg:not([class*='size-'])]:size-4 shrink-0 [&_svg]:shrink-0 outline-none focus-visible:border-ring focus-visible:ring-ring/50 focus-visible:ring-[3px] aria-invalid:ring-destructive/20 dark:aria-invalid:ring-destructive/40 aria-invalid:border-destructive",
  {
    variants: {
      variant: {
        default: 'bg-primary text-primary-foreground hover:bg-primary/90',
        destructive:
          'bg-destructive text-white hover:bg-destructive/90 focus-visible:ring-destructive/20 dark:focus-visible:ring-destructive/40 dark:bg-destructive/60',
        outline:
          'border bg-background shadow-xs hover:bg-accent hover:text-accent-foreground dark:bg-input/30 dark:border-input dark:hover:bg-input/50',
        secondary:
          'bg-secondary text-secondary-foreground hover:bg-secondary/80',
        ghost:
          'hover:bg-accent hover:text-accent-foreground dark:hover:bg-accent/50',
        link: 'text-primary underline-offset-4 hover:underline',
      },
      size: {
        default: 'h-9 px-4 py-2 has-[>svg]:px-3',
        sm: 'h-8 rounded-md gap-1.5 px-3 has-[>svg]:px-2.5',
        lg: 'h-10 rounded-md px-6 has-[>svg]:px-4',
        icon: 'size-9',
        'icon-sm': 'size-8',
        'icon-lg': 'size-10',
      },
    },
    defaultVariants: {
      variant: 'default',
      size: 'default',
    },
  },
)

function Button({
  className,
  variant,
  size,
  asChild = false,
  ...props
}: React.ComponentProps<'button'> &
  VariantProps<typeof buttonVariants> & {
    asChild?: boolean
  }) {
  const Comp = asChild ? Slot : 'button'

  return (
    <Comp
      data-slot="button"
      className={cn(buttonVariants({ variant, size, className }))}
      {...props}
    />
  )
}

export { Button, buttonVariants }
```


DEPENDENCIAS
------------

1. @radix-ui/react-slot
   - Permite composición avanzada
   - asChild prop para renderizar como hijo

2. class-variance-authority (CVA)
   - Sistema de variantes type-safe
   - Genera clases CSS basadas en props

3. @/lib/utils (cn function)
   - Merge de clases Tailwind
   - Elimina conflictos

PARTE 1: buttonVariants (CVA)
------------------------------

```typescript
const buttonVariants = cva(
  "clases-base",
  {
    variants: { ... },
    defaultVariants: { ... }
  }
)
```

CLASES BASE (aplicadas a TODOS los botones):
```
inline-flex              - Flexbox inline
items-center             - Centra verticalmente
justify-center           - Centra horizontalmente
gap-2                    - Espacio entre items (8px)
whitespace-nowrap        - Texto no se envuelve
rounded-md               - Bordes redondeados
text-sm                  - Texto pequeño (14px)
font-medium              - Peso medio
transition-all           - Transiciones suaves
disabled:pointer-events-none  - Deshabilita clicks
disabled:opacity-50      - Opacidad reducida cuando disabled
[&_svg]:pointer-events-none   - SVGs no capturan clicks
[&_svg:not([class*='size-'])]:size-4  - SVGs 16x16 por defecto
shrink-0                 - No se encoge
[&_svg]:shrink-0         - SVGs no se encogen
outline-none             - Sin outline default
focus-visible:border-ring     - Borde en focus
focus-visible:ring-ring/50    - Ring en focus
focus-visible:ring-[3px]      - Ancho del ring
aria-invalid:ring-destructive/20  - Ring rojo si inválido
dark:aria-invalid:ring-destructive/40  - Ring en dark mode
aria-invalid:border-destructive        - Borde rojo si inválido
```


VARIANTES: variant
------------------

1. default (defecto)
   ```
   bg-primary                    - Fondo primario
   text-primary-foreground       - Texto contraste
   hover:bg-primary/90           - Fondo al hover (90% opacidad)
   ```
   
   USO:
   ```tsx
   <Button>Click me</Button>
   <Button variant="default">Click me</Button>
   ```

2. destructive (acciones destructivas: delete, remove)
   ```
   bg-destructive                         - Fondo rojo
   text-white                             - Texto blanco
   hover:bg-destructive/90                - Hover más oscuro
   focus-visible:ring-destructive/20      - Ring rojo
   dark:focus-visible:ring-destructive/40 - Ring en dark
   dark:bg-destructive/60                 - Fondo en dark mode
   ```
   
   USO:
   ```tsx
   <Button variant="destructive">Delete</Button>
   ```

3. outline (botón con borde)
   ```
   border                          - Borde
   bg-background                   - Fondo default
   shadow-xs                       - Sombra pequeña
   hover:bg-accent                 - Hover con accent
   hover:text-accent-foreground    - Texto hover
   dark:bg-input/30                - Fondo dark mode
   dark:border-input               - Borde dark mode
   dark:hover:bg-input/50          - Hover dark mode
   ```
   
   USO:
   ```tsx
   <Button variant="outline">Cancel</Button>
   ```

4. secondary (botón secundario)
   ```
   bg-secondary                - Fondo secundario
   text-secondary-foreground   - Texto secundario
   hover:bg-secondary/80       - Hover más claro
   ```
   
   USO:
   ```tsx
   <Button variant="secondary">Secondary Action</Button>
   ```

5. ghost (sin fondo, solo hover)
   ```
   hover:bg-accent                  - Fondo solo en hover
   hover:text-accent-foreground     - Texto en hover
   dark:hover:bg-accent/50          - Hover dark mode
   ```
   
   USO:
   ```tsx
   <Button variant="ghost">Ghost Button</Button>
   ```

6. link (parece link, no botón)
   ```
   text-primary            - Color de link
   underline-offset-4      - Offset del underline
   hover:underline         - Underline en hover
   ```
   
   USO:
   ```tsx
   <Button variant="link">Learn more</Button>
   ```


VARIANTES: size
---------------

1. default (tamaño por defecto)
   ```
   h-9               - Altura 36px
   px-4              - Padding horizontal 16px
   py-2              - Padding vertical 8px
   has-[>svg]:px-3   - Padding reducido si tiene icono

2. sm (pequeño)
   ```
   h-8                  - Altura 32px
   rounded-md           - Bordes redondeados
   gap-1.5              - Gap reducido (6px)

3. lg (grande)

4. icon (botón cuadrado para solo icono)

LÓGICA DEL COMPONENTE
----------------------

```typescript
const Comp = asChild ? Slot : 'button'
```

COMPORTAMIENTO:
- Si asChild = false → renderiza <button>
- Si asChild = true → renderiza como hijo usando Slot

EJEMPLO asChild:
```tsx
// Sin asChild (renderiza button)
<Button>Click</Button>
→ <button>Click</button>

// Con asChild (renderiza como Link)
<Button asChild>
  <Link href="/about">About</Link>
</Button>
→ <a href="/about">About</a>  (con estilos de Button)
```


RENDER
------
```typescript
return (
  <Comp
    data-slot="button"
    className={cn(buttonVariants({ variant, size, className }))}
    {...props}
  />
)
```

PROCESO:
1. buttonVariants() genera clases CSS según variant y size
2. cn() merge las clases con className personalizada
3. Spread ...props agrega props adicionales
4. data-slot="button" para debugging/testing


EXPORTS
-------
```typescript
export { Button, buttonVariants }
```

Exporta:
- Button: componente
- buttonVariants: función CVA (para crear variantes custom)
