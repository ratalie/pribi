C√ìDIGO COMPLETO:

PROP√ìSITO:

Combina clases CSS de forma inteligente, manejando:
- Clases condicionales
- Conflictos de Tailwind
- Arrays y objetos de clases

   - Evita duplicados


FUNCIONAMIENTO:
---------------

PASO 1 - clsx():
```typescript

// ‚Üí "bg-red-500 text-white font-bold" (si condition es true)
```

PASO 2 - twMerge():


CASOS DE USO:
-------------

1. CLASES CONDICIONALES:

5. OBJETOS DE CLASES:

SIN cn():
